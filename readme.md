# 💊 智能健康助手

> 由5名大二学生共同开发的智能健康管理平台

## 📖 项目简介

智能健康助手是一个专为大学生设计的健康管理Web应用，旨在通过AI技术帮助人民更好地管理健康生活。

**计划功能：**
- 健康数据记录与追踪
- 个性化健康建议
- 运动与饮食规划
- 数据可视化分析
